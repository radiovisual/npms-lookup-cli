# npms-lookup-cli

> Search npm packages on npms.io from the command line.


## Install

```
$ npm install --global npms-lookup-cli
```


## Usage

```
$ npms --help

  Usage
    npms [input]

  Examples
    $ npms object-assign
    $ npms express
```

## Example

Look up the [`object-assign`](https://www.npmjs.com/package/object-assign) package on npms.io from the command line:

```
$ npms object-assign
```
